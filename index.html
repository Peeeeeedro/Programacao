Principais Comandos para Criar um Site

1. Estrutura Básica de um Documento HTML

O HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. A estrutura básica de um documento HTML inclui as tags <!DOCTYPE html>, <html>, <head>, <title> e <body>. Cada uma dessas tags tem um papel específico na definição do conteúdo e da estrutura da página.

2. Adicionando Estilos com CSS

O CSS (Cascading Style Sheets) é usado para controlar a aparência do conteúdo HTML. A sintaxe básica do CSS envolve a seleção de elementos HTML e a aplicação de propriedades e valores. Por exemplo, body { color: black; } define a cor do texto de todo o corpo da página como preto.

3. Inserindo Imagens

Para inserir imagens em uma página HTML, utilizamos a tag <img>. A tag <img> requer o atributo src, que especifica o caminho da imagem, e o atributo alt, que fornece uma descrição alternativa da imagem. Exemplo: <img src="caminho/para/imagem.jpg" alt="Descrição da Imagem">.

4. Criando Links

Para criar links para outras páginas ou sites, usamos a tag <a>. O atributo href define o URL para o qual o link aponta. Exemplo: <a href="https://www.example.com">Visite o Example.com</a>.
